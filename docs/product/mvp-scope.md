# Chat-Net MVP Scope

## Muss enthalten

- 1:1 Chat
- Gruppenchat / Channels
- Datei-Upload
- Lesebestätigungen
- Tippindikator
- Suche in Nachrichten
- Push-Benachrichtigungen
- Accounts mit Profilbild
- Moderner, übersichtlicher, vertrauenswürdiger Look

## Auth

- Google OAuth Login
- E-Mail + Passwort Login
- Registrierung per E-Mail + Passwort
- E-Mail Verifizierung
- Passwort vergessen / Zurücksetzen

## Plattformen

- Web-App
- iOS/Android App

## Nicht im aktuellen Code vollständig umgesetzt

- Persistente Chat-Nachrichten (derzeit API-Scaffold)
- Upload, Suche, Push als volle Features (derzeit vorbereitet)
- Produktive Google-Token-Verifizierung
